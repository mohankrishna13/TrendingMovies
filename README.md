# TrendingMovies

<b>Libraries & Dependencies:</b>

1.Glide

2.Retrofit

3.Dagger Hilt

4.Kotlin-coroutines

5.Circleimageview

<b>Architecture:</b> MVVM

1.In Home Screen(First screen) user can search Movies by using search option in toolbar.The movies will be fetched from api.

2.Once Response is succes the data will be shown in two Recyclerview(Horizontal , vertical).

3.If user selects any movie from any one of the recyclerview the screen will be navigated into Second Screen(Details Screen).

4.In the Details screen user can find the details of the selected movie.
